# 客观题题库练习网站

这是一个直接读取 `questions.json` 的静态题目练习网站。题目会随机排列，并优先完整遍历尚未做过的题目；未做题遍历完后，再随机复习已做题。

## 启动

浏览器出于安全限制，不能通过双击 `index.html` 读取本地 JSON。请在本目录打开终端并运行：

```powershell
python -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 功能

- 从 `questions.json` 加载并校验题库
- 未做题优先的随机练习，单轮内不重复
- 单选、多选、判断题自动判题
- 练习进度、正确率与错题记录持久化
- 错题专项练习、错题清空与文本导出
- 全部练习进度可导出为 JSON，并在其他浏览器或设备导入

## GitHub Pages

站点可直接从仓库根目录发布，无需构建步骤。进入仓库的 **Settings → Pages**，选择 **Deploy from a branch**，并将 `main` 分支的根目录设为发布源。
